## Quick Start

<CodeGroup>

```bash cURL
curl --location 'https://api2.matterai.so/v1/web/chat/completions' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer MATTERAI_API_KEY' \
--data '{
    "model": "axon-2-5-pro",
    "max_new_tokens": 2048,
    "temperature": 0.1,
    "messages": [
        {
            "role": "user",
            "content": "Hi"
        }
    ],
    "stream": false
}'
```

```javascript JavaScript
import OpenAI from "openai";

const client = new OpenAI({
  apiKey: "MATTERAI_API_KEY",
  baseURL: "https://api2.matterai.so/v1",
});

const response = await client.chat.completions.create({
  model: "axon-2-5-pro",
  messages: [{ role: "user", content: "Hi" }],
  max_tokens: 2048,
  temperature: 0.1,
});

console.log(response.choices[0].message.content);
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="MATTERAI_API_KEY",
    base_url="https://api2.matterai.so/v1"
)

response = client.chat.completions.create(
    model="axon-2-5-pro",
    messages=[
        {"role": "user", "content": "Hi"}
    ],
    max_tokens=2048,
    temperature=0.1
)

print(response.choices[0].message.content)
```
