## Prompt Caching

Cached input tokens are automatically discounted at 50% off the standard input price. Caching kicks in automatically for repeated prompt prefixes.

| Model         | Cached Input Price (per 1M tokens) |
| ------------- | ---------------------------------- |
| Axon 2.5 Pro  | $1.00                              |
| Axon 2.5 Mini | $0.25                              |
